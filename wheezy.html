Wheezy Web

wheezy.web is a lightweight, high performance, high concurrency WSGI web framework with the key features to build modern, efficient web

It is optimized for performance, well tested and documented.

Wheezy is completely light weight, Built for Performance

Encourage separation of code and content. Work more productively at higher level of abstraction.

Runs on Python
The application works on Python 2.4-2.7 or 3.2+.
Puse-based MVC
MVC with complete push-based architectural pattern. Similar to Django, Wheezy, Ruby on Rails, Symfony, Yii, Spring MVC, Stripes, Play, CodeIgniter and Struts2
Functionalities
Most light weight Routing, model update/validation, authentication/authorization, content caching with dependency, xsrf/resubmission protection, AJAX+JSON, i18n (gettext), middlewares, and many many many more.
Template engine agnostic
Integration with template engines like jinja2, mako, tenjin and wheezy.template. BTW, it also support widgets for all the template engines.

Completely Open Source.

Code base is hosted on BitBucket. We also use it for issue tracking. You can use the Freenode IRC #wheezy.web for any kind of help.
